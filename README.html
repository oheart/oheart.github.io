用hexo-blog搭建的个人博客。

英文doc

初始化hexo-blog

  1. npm install hexo-cli -g
  2. hexo init blog
  3. cd blog
  4. npm install
  5. hexo server

    常见用法

  6. 新建文章： hexo new 文章名
  7. 新建页面：hexo new page about
    用localhost:4000/about/index.html访问
  8. 修改项目根目录的_config.yml中的 post_asset_folder 参数设置文章的资源文件夹要不要.默认为false,设置为true新建文章时可以生成对应的文件夹
    资源文件夹
  9. 部署（deploy）
  • github新建oheart.github.io的仓库，这里的oheart是用户名。
  • doc
  • hexo generate - d 文件生成后立即部署网站

编辑文章内容

  1. title: 标题
  2. date: 创建日期
  3. tags: 标签
  4. categories: 分类

    scaffolds里面的模板(md文件)

  5. post.md（新建文章的模板）
    给post.md添加categories字段，那么每次新建文章时就会多出来一个categories字段.
  6. page.md（新建页面的模板）
